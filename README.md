# AES Encrypted Chat Application

## Overview

This project is a Java-based encrypted chat application that uses AES (Advanced Encryption Standard) for secure communication between a client and a server using Java Socket Programming.

## Features

* AES Encryption and Decryption
* Client-Server Communication
* Java Socket Programming
* Secure Message Transfer

## Technologies Used

* Java (JDK 21)
* Visual Studio Code
* Java Cryptography (AES)
* Java Networking (Sockets)

## Project Files

* AESUtil.java
* Server.java
* Client.java

## How to Run

Compile the project:

```bash
javac *.java
```

Run the server:

```bash
java Server
```

Open another terminal and run the client:

```bash
java Client
```

## Output

* Client encrypts the messages.
* Server receives encrypted messages.
* Server decrypts and displays the original messages.
